# Snatchd Admin Dashboard

Next.js admin dashboard for managing Snatchd products, stores, and orders.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deployment

This app is configured for Vercel deployment. Make sure to set the **Root Directory** to `snatchd-admin` in Vercel project settings.

## Environment Variables

No environment variables required - Firebase config is included in the codebase.
